# Cross-Product Adaptation Guide
<!-- date: 2026-03 -->

How to take a script written for one product and adapt it for another. Used in **Mode 1 (Multi-Product Adaptation)**.

---

## When to Use

- You have a **proven script** (high engagement, good conversion) for Product A
- You want to run the same script structure for Product B
- The pattern and Schwartz framework stay the same — only product-specific details change

**Prerequisite**: Identify which pattern the source script uses (read `directory.md`), then read that pattern file before adapting.

---

## What Changes vs. What Stays

| Stays the Same (Structure) | Changes (Content) |
|---------------------------|-------------------|
| Hook formula (3 wrong answers, symptom list, etc.) | Product name and brand name |
| Number of sections and sentence count | Ingredient names and aliases |
| Schwartz pattern (D stays D, A stays A) | Mechanism explanation (how it works) |
| Emotional arc and pacing | Specific symptoms targeted |
| Transition phrases ("But here's the thing...") | Wrong answers in misdirection hooks |
| CTA structure (orange button, scarcity) | Price anchor and deal structure |
| Urgency triggers (sold out, months to restock) | Heritage/origin story |
| Social proof positioning | Benefit stack (mapped to new product) |
| Tribal identity angle (if applicable) | Sensory language (taste, texture, experience) |
| Rhetorical questions and emphasis patterns | Dosage and usage instructions |

## Mechanism Mapping

The **mechanism** is the "how it works" explanation — the scientific-sounding reason the product solves the problem. When adapting, map the source product's mechanism to the target product's equivalent.

### Mapping Process

1. **Identify the source mechanism**: What biological process does the original script blame?
2. **Find the target equivalent**: What does the new product actually address?
3. **Match the complexity level**: If the source uses simple language ("it calms your stress"), keep it simple. If it uses technical terms ("nitric oxide production"), use equivalent technical terms.
4. **Preserve the villain-hero arc**: Source has a villain (cortisol, inflammation, toxins) → new product needs its own villain that the ingredient defeats.

### Saffron vs. Rhodiola Mechanism Map

| Element | Rhodiola (Salvora / Floraviva / Flor Viva) | Saffron (Rosabella) |
|---------|---------------------------------------------|---------------------|
| **Core mechanism** | Cortisol reduction — lowers chronic stress hormone | Serotonin support — boosts mood-regulating neurotransmitter |
| **Villain** | Cortisol ("your body's stress hormone stuck on overdrive") | Hormonal imbalance / inflammation |
| **How it works (simple)** | Calms the nervous system, shifts body out of fight-or-flight | Supports natural serotonin levels, calms the nervous system |
| **How it works (technical)** | Helps body produce nitric oxide → relaxes blood vessels → improves circulation | Supports healthy serotonin levels → reduces inflammation → regulates mood |
| **Category** | Adaptogen | Natural spice / superfood |
| **Heritage** | "Ancient remedy" / "golden root" | "Used for thousands of years in the Mediterranean" |
| **Key ingredient** | Rhodiola Rosea | Saffron extract (sometimes + L-theanine + magnesium glycinate) |
| **Format** | Capsules (2 pills daily) | Powder / capsules (1 teaspoon or capsule daily) |
| **Action verbs** | "Take" / "one capsule in the morning" | "Add to your routine" / "one dose" / "one teaspoon" |

## Symptom Mapping

Different products target overlapping but distinct symptom sets. When adapting, swap symptoms to match the new product's documented benefits.

| Symptom Category | Rhodiola Symptoms | Saffron Symptoms |
|-----------------|-------------------|------------------|
| **Weight / body** | Belly weight, bloating, midsection fat, jeans feeling tighter | Bloating, midsection weight, body feeling heavy |
| **Energy** | Fatigue, feeling worn down, no energy | Fatigue, feeling drained, low energy |
| **Mental** | Brain fog, poor focus, mental fatigue | Brain fog, forgetfulness, distraction, lack of clarity |
| **Skin / appearance** | Facial puffiness, thinning hair, loose/sagging skin, looking older | Dull skin, looking older, loss of glow |
| **Sleep** | Poor sleep, restlessness | Interrupted sleep, night sweats, poor sleep |
| **Mood** | Overwhelm, stress, feeling stuck | Mood swings, anxiety, irritability, feeling vulnerable, "can't find yourself" |
| **Hormonal** | General stress response | Hot flashes, cold flashes, temperature changes, dryness, menopause symptoms |
| **Physical** | Stiffness, achiness, tired feeling | Cramps, achiness, circulation issues |

**Key difference**: Rhodiola scripts lean on **stress/cortisol** as the universal villain. Saffron scripts lean on **hormonal imbalance/menopause** as the villain. Both share overlapping symptoms (fatigue, brain fog, weight) but frame the *cause* differently.

## Adaptation Checklist (Mode 1)

Step-by-step process for adapting a script to a new product:

### Before Writing

- [ ] Identify the source script's pattern (check `directory.md`)
- [ ] Read the pattern file for Schwartz breakdown
- [ ] Read the target product's documentation (`reference/products/`)
- [ ] Map the source mechanism → target mechanism (use table above)
- [ ] Map the source symptoms → target symptoms

### During Adaptation

- [ ] Swap all product names and brand names
- [ ] Swap ingredient names and aliases ("golden root" → "ancient spice")
- [ ] Replace mechanism explanation with target product's mechanism
- [ ] Replace symptom lists with target product's relevant symptoms
- [ ] Update "wrong answers" in misdirection hooks (must be competitors/alternatives relevant to the NEW product's category)
- [ ] Update heritage/origin story to match new product
- [ ] Update dosage and usage instructions
- [ ] Update sensory language (how it tastes, feels, is consumed)
- [ ] Update price anchor and deal structure from product docs
- [ ] Update brand story (small company, black-owned, etc.) if applicable

### After Adaptation

- [ ] Verify sentence count matches original
- [ ] Verify hook structure is preserved exactly
- [ ] Verify CTA structure is preserved
- [ ] Verify all product claims match the product documentation — no invented benefits
- [ ] Run TikTok compliance check (Mode 1 Step 3)
- [ ] Check that NO traces of the original product remain (search for old brand name, ingredient, mechanism terms)

## Common Pitfalls

1. **Leftover traces**: Search-and-replace isn't enough. "Cortisol" might appear as "your stress hormone" — catch the paraphrases too.
2. **Wrong-answer mismatch**: If the source script says "ain't green tea, ain't turmeric" for Rhodiola, the Saffron version needs wrong answers relevant to saffron's category (e.g., "ain't maca powder, ain't ashwagandha").
3. **Mechanism copy-paste**: Don't just swap the ingredient name in the mechanism paragraph. The *explanation* of how it works must change (cortisol reduction ≠ serotonin support).
4. **Symptom bleed**: Don't keep symptoms from the source product that the target product doesn't address. Check the product docs.
5. **Heritage mismatch**: African ancestral roots work for Rhodiola's demographic targeting. Saffron uses Mediterranean heritage. Don't mix them.
6. **Price/deal mismatch**: Each brand has its own pricing structure. Update the price anchor, deal, and bottle count.
