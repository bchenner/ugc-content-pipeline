# Pattern B: Command Structure

---

## PATTERN B: "IF/THEN TAKE PRODUCT" COMMAND STRUCTURE

**Examples:** Scripts #6, #28, #29

**Core Structure:**
```
If [problem], take [product].
If [problem], take [product].
If [problem], take [product].
(Repeat 6-8 times)
```

**Schwartz Principle:** Commanding Authority + Simplification

### PATTERN B ANALYSIS:

**Why This Works:**

1. **Hypnotic Repetition:** "Take Rhodiola Rosea" appears 7 times in first 7 lines
2. **Authority Through Command:** Not asking, TELLING
3. **Covers All Objections:** Lists every possible problem upfront
4. **Simplification:** Eliminates decision fatigue
5. **Direct Action:** No story, no buildup—just DO THIS

**Schwartz Principle Applied:**
This is **Stage 4-5 sophistication** where the market is exhausted from complex solutions. The repetition creates hypnotic authority.
